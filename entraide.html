Questions fréquentes

Tout ce qu'il faut savoir avant de rejoindre l'association ou d'assister à un atelier.

La porte d'entrée principale est notre serveur Discord — c'est là que tout se passe : annonces d'ateliers, discussions, demandes d'aide. Il suffit de rejoindre le serveur, de vous présenter dans le canal dédié et le tour est joué. Aucune cotisation ni adhésion formelle n'est requise pour participer.
Non, tous nos ateliers sont entièrement gratuits. L'association est à but non lucratif (loi 1901) et repose sur le bénévolat. Notre objectif est de rendre l'apprentissage numérique accessible à toutes et à tous dans le département de l'Eure, sans barrière financière.
Les ateliers sont programmés de façon irrégulière selon les disponibilités des bénévoles et les propositions des membres. Les annonces sont publiées sur le serveur Discord en avance. Pour ne rien manquer, activez les notifications du canal #annonces.
Absolument ! Nos ateliers s'adressent à tous les niveaux, du grand débutant au développeur confirmé. Chaque session précise le niveau recommandé.

Niveaux habituellement disponibles : Débutant Intermédiaire Avancé

N'hésitez pas à poser vos questions dans les canaux Discord — la communauté est bienveillante.
Vous avez une compétence à partager ou un sujet que vous aimeriez voir abordé ? Utilisez le formulaire de proposition d'atelier sur la page d'accueil, ou signalez-vous directement sur le Discord. Aucune expertise particulière n'est nécessaire pour animer — l'envie de partager suffit !
Les deux ! Certains ateliers se tiennent en ligne (Discord vocal ou visio), d'autres en présentiel dans le département de l'Eure. Le format est précisé dans chaque annonce. Les sessions en ligne permettent à toute personne, où qu'elle soit en Normandie (ou ailleurs), de participer sans se déplacer.
Le code du site est publié sous licence MIT sur github.com/aleuredigitale. Pour contribuer : forkez le dépôt, créez une branche, ouvrez une Pull Request en décrivant vos changements. Les contributions — corrections, améliorations, nouveaux contenus — sont les bienvenues !
